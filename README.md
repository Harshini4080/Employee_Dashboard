# Employee Dashboard – Jotish Assignment

This project is developed as part of the **Jotish assignment**.  
It is a React-based Employee Dashboard application that displays employee data with visualization and interactive features.

---

##  Features

- Login Page
- Employee List (API Integration using Axios)
- Salary Bar Chart (Top 10 Highest Paid Employees)
- Employee Locations Map (Leaflet Map Integration)
- Camera Capture in Employee Details Page
- Navigation using React Router
- Clean and responsive UI

---

##  Tech Stack

- React (Vite)
- React Router DOM
- Axios (API Integration)
- Chart.js (Data Visualization)
- React Leaflet (Map Visualization)
- CSS
